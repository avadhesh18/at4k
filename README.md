# AT4K Launcher for Android TV
A powerful, privacy friendly and clean launcher for Android TV that transforms your viewing experience with enhanced app management and personalized home screen layouts. Inspired by Apple TV's approach to a simple, no-nonsens, ad-free UI.

## 🔧 Installation

1. Download AT4K Launcher from the Google Play Store
2. Open the app and grant necessary permissions
4. Set AT4K as your default launcher if prompted, if not then use a tool like Launcher Manager

## 📖 Usage

### Setting as Default Launcher

1. Open Android TV Settings
2. Navigate to **Apps** → **Default Apps** → **Home App**
3. Select **AT4K Launcher**
4. If this option does not exist in your Android TV or Amazon Fire TV then use tools like Launcher Manager to get it working.

### Customizing Your Home Screen

- **Favoruites**: Moves apps to favourites to view their watch next and channels programmes.
- **Organize Layout**: Drag and drop apps to rearrange
- **Customize Appearance**: Adjust themes and visual settings in the app settings

## 🎯 Compatible Devices

AT4K Launcher works on:
- Android TV boxes
- Smart TVs running Android TV / Google TV
- Nvidia Shield TV
- Xiaomi Mi Box
- Fire TV devices
- Any device running Android TV 8+
